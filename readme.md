# Документация тестов проекта Island Simulation

Данный документ содержит описание набора unit-тестов для проекта симуляции острова с животными. Тесты написаны с использованием **JUnit 5** и покрывают базовую функциональность модели, создание животных, а также сложные сценарии взаимодействия (охота, конкуренция, пищевые цепочки).

---

## Цель тестирования

Тесты обеспечивают проверку:
- Корректности создания острова и ячеек;
- Правильности создания животных (имя, вес, состояние жизни);
- Наследования и полиморфизма (хищники и травоядные);
- Логики охоты и пищевых предпочтений;
- Сценариев с несколькими хищниками и поедания травоядными.

---

## Структура тестов

### 1. Базовые тесты инфраструктуры
| Класс | Описание |
|-------|----------|
| `MainTest` | Проверяет, что метод `main` запускается без исключений. |
| `IslandTest` | Проверяет создание острова с заданными размерами. |
| `IslandCellTest` | Проверяет создание ячейки с координатами. |

### 2. Абстрактные тесты животных
| Класс | Описание |
|-------|----------|
| `AnimalTest` | Абстрактный тест, содержащий общие проверки для всех животных (создание, смерть). |
| `HerbivoreTest` | Проверяет, что травоядное является экземпляром `Animal`. |
| `PredatorTest` | Проверяет, что хищник является экземпляром `Animal`. |

### 3. Тесты конкретных хищников
| Класс | Описание |
|-------|----------|
| `BoaTest` | Проверяет имя и вес удава. |
| `EagleTest` | Проверяет имя и вес орла. |
| `FoxTest` | Проверяет имя и вес лисы. |
| `WolfTest` | Проверяет имя и вес волка. |
| `BearTest` | Проверяет имя и вес медведя. |

### 4. Тесты конкретных травоядных
| Класс | Описание |
|-------|----------|
| `BuffaloTest` | Проверяет имя и вес буйвола. |
| `CaterpillarTest` | Проверяет имя и вес гусеницы. |
| `DeerTest` | Проверяет имя и вес оленя. |
| `DuckTest` | Проверяет имя и вес утки. |
| `GoatTest` | Проверяет имя и вес козы. |
| `HorseTest` | Проверяет имя и вес лошади. |
| `MouseTest` | Проверяет имя и вес мыши. |
| `RabbitTest` | Проверяет имя и вес кролика. |
| `SheepTest` | Проверяет имя и вес овцы. |
| `BoarTest` | Проверяет имя и вес кабана. |

### 5. Сценарии взаимодействия
| Класс | Описание |
|-------|----------|
| `HuntingScenarioTest` | Содержит три сложных теста:<br>• `testWolfHuntsRabbitAndMouseInSameCell` – волк охотится в одной ячейке на кролика и мышь; проверяется увеличение веса волка и уменьшение числа живых жертв.<br>• `testMultiplePredatorsCompetition` – волк и лиса конкурируют за кролика; проверяется, что только один хищник смог поесть.<br>• `testHerbivoreChainEating` – утка и мышь поедают гусеницу; проверяется, что гусеница съедена или оба животных изменили вес. |

## Заключение

Тесты покрывают ключевую логику проекта и могут служить документацией ожидаемого поведения классов. При добавлении новых животных или изменении логики взаимодействия рекомендуется дополнять соответствующие тесты.
